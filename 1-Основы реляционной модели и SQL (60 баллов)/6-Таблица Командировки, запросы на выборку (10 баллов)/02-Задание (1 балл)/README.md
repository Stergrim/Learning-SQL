**Задание**

Вывести в алфавитном порядке фамилии и инициалы тех сотрудников, которые были в командировке в Москве.

Введите SQL запрос

*Результат:*

```mysql

+---------------+

| Баранов П.Е.  |
| Колесов С.П.  |
| Лебедев Т.К.  |
| Семенов И.В.  |
+---------------+

```

```mysql
SELECT name
FROM trip
WHERE city = 'Москва'
GROUP BY name
ORDER BY name;
```
